## Stack

Nesse projeto, usaremos Java com Spring Boot Framework.

## Instruções para execução

### Requisitos

- Java 17
- Maven

### Passo a passo

Execute o seguinte comando no terminal

```
cd jbank
./mvnw spring-boot:run
```

Obs.: caso esteja usando windows troque `./mvnw` por `nvnw.cmd`
